# Taegeuk Beats - Rhythm Action Game

## Deskripsi
Taegeuk Beats adalah game Rhythm Action + QTE Combat berbasis naratif yang menceritakan perjalanan siswa Taekwondo. Proyek ini dibuat menggunakan Unity 2022.3 LTS.

## Fitur Utama (Fase 2)
1. **Rhythm Gameplay:** Mekanik pukul not (F/J) dengan penilaian akurasi (Perfect/Miss).
2. **QTE System:** Quick Time Event dengan timer visual melingkar.
3. **Exploration Mode:** Pergerakan Top-Down 2.5D untuk eksplorasi area cerita.
4. **Game Loop:** Sistem Menu, Pause, Game Over, dan Restart level.
5. **UI System:** HUD Skor, Combo, HP Bar, dan Damage Popup text.

## Cara Install / Play
1. Clone repository ini.
2. Buka dengan Unity Hub (Versi 2022.3.13f1).
3. Buka Scene `MainMenu` di folder `Assets/Scenes`.
4. Tekan Play.

## Kontrol
- **Exploration:** WASD / Arrow Keys (Gerak).
- **Rhythm:** Tombol [F] dan [J] (Pukul).
- **UI:** Tombol [Esc] (Pause).
