# Taegu - The Legend of Rhythm

**Taegu - The Legend of Rhythm** adalah game ritme 2D sederhana yang dikembangkan sebagai Tugas Akhir mata kuliah Pemrograman Berorientasi Objek (PBO). Pemain harus menekan tombol sesuai irama musik untuk mengalahkan hantu yang mendekat.

---

## Cara Bermain (User Manual)

1. **Start Game:** Klik tombol "Play" di Menu Utama.
2. **Gameplay:**
   - Perhatikan not balok yang berjalan dari kanan ke kiri.
   - Tekan tombol **F** untuk memukul not **Biru** (Memukul).
   - Tekan tombol **J** untuk memukul not **Merah** (Menendang).
   - Pukul tepat saat not berada di garis sensor (lingkaran target).
3. **Winning & Losing:**
   - **Combo:** Pukul berturut-turut untuk meningkatkan skor.
   - **HP:** Darah akan berkurang jika meleset (Miss). Game Over jika HP = 0.
   - **Rank:** Selesaikan lagu dengan skor tinggi untuk mendapat Rank S/A/B.
- **UI:** Tombol [Esc] / II symbol (Pause Game).
---

## Cara Install / Play
1. Clone repository ini.
2. Buka dengan Unity Hub (Versi 2022.3.13f1).
3. Buka Scene `MainMenu` di folder `Assets/Scenes`.
4. Tekan Play.
